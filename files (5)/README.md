# 🌍 Sustainable Development & Climate Action Website

A comprehensive, multi-technology website dedicated to promoting sustainable development and climate action.

## Project Overview

This website combines education, data visualization, community engagement, and interactive tools to drive climate awareness and action.

## 🛠 Technology Stack

### Frontend
- **Next.js 14** - React framework
- **React 18** - UI library
- **Tailwind CSS** - Styling
- **Framer Motion** - Animations
- **Chart.js** - Data visualization
- **D3.js** - Advanced charts
- **Leaflet** - Interactive maps

### Backend
- **Node.js** - Runtime
- **Express.js** - Web framework
- **PostgreSQL** - Database
- **Redis** - Caching
- **Prisma** - ORM

### DevOps & Tools
- **Docker** - Containerization
- **GitHub Actions** - CI/CD
- **Jest** - Testing
- **TypeScript** - Type safety

## 🚀 Quick Start

### Prerequisites
- Node.js 18+
- PostgreSQL 13+
- Docker (optional)

### Installation

```bash
# Clone repository
git clone https://github.com/raju-bot-74/sustainable-climate-action.git
cd sustainable-climate-action

# Install frontend dependencies
cd frontend
npm install
npm run dev

# In another terminal - Install backend
cd backend
npm install
npm run dev
