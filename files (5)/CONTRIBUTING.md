# Contributing to Sustainable Climate Action

We're thrilled you want to contribute! This document provides guidelines for contributing.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/sustainable-climate-action.git`
3. Create a feature branch: `git checkout -b feature/amazing-feature`
4. Make your changes
5. Push to your fork: `git push origin feature/amazing-feature`
6. Open a Pull Request
