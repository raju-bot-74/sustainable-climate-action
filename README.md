# Sustainable Development & Climate Action Website

A comprehensive, multi-technology website dedicated to promoting sustainable development and climate action.

## 🌍 Project Overview

This website combines education, data visualization, community engagement, and interactive tools to drive climate awareness and action.

## 🛠 Technology Stack

### Frontend
- **React.js** - Interactive UI components
- **Next.js** - Server-side rendering and static generation
- **Tailwind CSS** - Responsive styling
- **D3.js / Chart.js** - Data visualization
- **Leaflet.js** - Interactive maps
- **Framer Motion** - Animations

### Backend
- **Node.js** - Runtime environment
- **Express.js** - Web server framework
- **PostgreSQL** - Database
- **Prisma ORM** - Database management
- **Redis** - Caching

### Additional Tools
- **GitHub Pages / Vercel** - Hosting
- **GitHub Actions** - CI/CD
- **Jest & Testing Library** - Testing
- **ESLint & Prettier** - Code quality
- **Docker** - Containerization
